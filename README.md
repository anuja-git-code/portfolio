# Anuja's Portfolio

A modern, interactive portfolio website showcasing skills, projects, and achievements.

## Features

- Responsive design
- Dark/Light mode toggle
- Smooth scrolling navigation
- Typing animation
- Interactive skill bars
- Expandable education table
- Project details toggle
- Certificate gallery with modal
- Back to top button
- Animated elements on scroll

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- JavaScript (ES6+)
- Google Fonts (Poppins)

## Required Images

Please add the following images to the root directory:

1. `profile.jpg` - Profile picture for the home section (recommended: 400x400px, square)
2. `about.jpg` - Image for the about section (recommended: 300x300px)
3. `project1.jpg` - Image for the Gym Scheduling System project
4. `project2.jpg` - Image for the Portfolio Website project
5. `hackathon.png` - Hackathon certificate image
6. `cpp-cer.jpeg` - Capstone project certificate image
7. `cert3.png` - Python bootcamp certificate image
8. `phone.svg` - Phone icon (SVG format)
9. `email.svg` - Email icon (SVG format)
10. `github.svg` - GitHub icon (SVG format)
11. `linkedin.svg` - LinkedIn icon (SVG format)

## How to Run

1. Clone or download the repository
2. Add the required images as mentioned above
3. Open `index.html` in a web browser
4. For live development, use a local server (e.g., Live Server extension in VS Code)

## Customization

- Edit `index.html` to update personal information
- Modify `style.css` to change colors, fonts, or layout
- Update `script.js` to add new interactive features

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge
